\section{Results}
The primary quantitative results come from C-MAPSS FD001. The adaptive looped model produced the strongest macro F1 score on the held-out test set while using far fewer internal steps than the fixed-depth loop.

\subsection{Primary Findings}
\begin{itemize}
    \item The adaptive model achieved test macro F1 of 0.9183, outperforming the fixed-depth loop at 0.8891 and the direct baseline at 0.8609.
    \item The adaptive model required an average of only 1.21 recurrent steps on the test set, compared with 6.0 steps for the fixed-depth loop.
    \item Relative to the fixed-depth loop, the adaptive policy reduced average loop computation by approximately 79.8\%.
    \item The adaptive model preserved strong critical-class performance, reaching perfect precision and 0.75 recall on the small critical test subset.
\end{itemize}

\subsection{Validation and Generalization}
On validation, the fixed-depth loop reached the best macro F1 score of 0.9366, while the adaptive model reached 0.9142 at an average depth of 1.39 steps. On the official test split, however, the adaptive model generalized better and delivered the strongest macro F1 score. This pattern suggests that fixed full-depth recurrence may fit the development split more aggressively, while adaptive early exit regularizes the model enough to improve test-time class balance.

\subsection{Secondary Dataset Status}
The IMS pipeline is implemented with extracted run-to-failure vibration snapshots, summary-statistic feature generation, and cached preprocessing. Early temporal-split runs on IMS showed degenerate validation folds with missing classes, so the final reported IMS results use stratified validation instead. Under that evaluation setup, the adaptive model performed strongly across the first, second, and fourth IMS runs. The main quantitative claims in this report remain anchored on C-MAPSS because it provides official train/test splits, but the project now shows results for all local dataset sources available in the repository.
