\section{Demo}

\begin{frame}{Demo Flow}
  \begin{itemize}
    \item Load a telemetry CSV or a prepared benchmark sample.
    \item Display health class, confidence, exit depth, and recommended action.
    \item Show which signals drifted most strongly in the selected window.
    \item Compare adaptive depth against the fixed-depth baseline on the same sample.
  \end{itemize}
\end{frame}

\begin{frame}{Dashboard Deliverable}
  \begin{itemize}
    \item Streamlit app reads the saved artifact bundles directly from `artifacts/`.
    \item Benchmark view shows metric comparisons, confusion matrices, and training curves.
    \item Sample inspector traces adaptive confidence and exit depth on a selected CMAPSS window.
    \item Custom CSV upload supports quick maintenance-style walkthroughs on user-provided telemetry.
  \end{itemize}
\end{frame}
