\section{Limitations}

\begin{frame}{Broader Impact \& Limitations}
  \begin{itemize}
    \item We learned that a task-specific looped model can fit sensor data better than a text-based LLM pipeline.
    \item But these are still benchmark-style experiments, not a live building or factory deployment.
    \item Some labels, especially actions and subsystems, are still approximations.
    \item The latency numbers are real measurements, but only on our local machine.
    \item The new HVAC extension is only partially head-to-head because the matching LLM runs were too slow on CPU.
  \end{itemize}
\end{frame}
