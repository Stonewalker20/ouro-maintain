\section{Limitations}

\begin{frame}{Current Limitations}
  \begin{itemize}
    \item CMAPSS is simulated telemetry, not a live maintenance deployment stream.
    \item IMS labels are implicit and temporal splits can omit one health class.
    \item The current system predicts health state only; action labels are still heuristic.
    \item Compute is measured by loop depth, not full latency profiling.
  \end{itemize}
\end{frame}
