\section{Experiments}

\begin{frame}{What We Tested}
  \begin{enumerate}
    \item Is the looped model more accurate than the LLM baselines?
    \item Does early stopping save time without hurting quality?
    \item Does the model use more steps on harder examples?
    \item Do the same patterns hold across the full dataset matrix?
  \end{enumerate}
\end{frame}

\begin{frame}{How We Compared the Models}
  \begin{itemize}
    \item We trained a simple baseline, a fixed loop, an adaptive loop, and two LLM baselines.
    \item The LLM baselines were a generic pretrained DistilBERT model and a task-specific DistilBERT model.
    \item We compared prediction quality and latency.
    \item We repeated that comparison across all local C-MAPSS and IMS subsets.
  \end{itemize}
\end{frame}
