# Presentation Bundle

This directory contains the bundled demo content used by the dashboard when you want a live presentation path with no setup.

## Contents

- `presentation_bundle.csv` - a compact labeled telemetry sample with three demo assets
- `presentation_bundle.json` - curated presentation annotations for health state, maintenance action, subsystem attribution, confidence, loop depth, and latency

## Use

The Streamlit dashboard can load this bundle directly from the `Presentation bundle` tab.
It is intentionally small so it can be checked into the repo and used during a talk without external downloads or a trained artifact tree.
